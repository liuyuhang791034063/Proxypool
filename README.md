## Proxypool

### 程序介绍：
模仿[开源代理池](https://github.com/Germey/ProxyPool)，写的自己的一个代理池

### 环境要求：
- python==3.6
- beautifulsoup4==4.6.1
- aiohttp==3.3.2
- requests==2.19.1
- redis==2.10.6  
列出重点使用的库，具体库可以通过requirement.txt安装

### 程序使用：
- 安装环境
- 配置setting文件
- 运行run.py即可开始运行程序
